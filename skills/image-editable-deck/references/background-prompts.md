# 無字背景提示規格

```text
Use case: scientific-educational
Asset type: 16:9 text-free slide background
Primary request: 本頁概念的視覺隱喻
Scene/backdrop: 場景與主體
Style/medium: 全份簡報共用風格
Composition/framing: 指定文字保留區
Color palette: 共用色彩
Constraints: no text, no letters, no numbers, no labels, no logo, no watermark
Avoid: 偽文字、假圖表、密集裝飾、文字區紋理過強
```

## 範例

```text
Use case: scientific-educational
Asset type: 16:9 text-free slide background
Primary request: 表現「多個證據共同判斷試題品質」
Scene/backdrop: 三個半透明圓環形成交集，旁邊有教育研究者觀察資料
Style/medium: 現代扁平插畫，正式教育研究風格
Composition/framing: 右側 55% 放視覺，左側 40% 保留平整深色空間供 PowerPoint 文字
Color palette: 米白、藍綠、橘色
Constraints: absolutely no text, no letters, no numbers, no labels, no logo, no watermark
Avoid: 偽文字、亂碼、假圖表、過度發光、密集背景
```
