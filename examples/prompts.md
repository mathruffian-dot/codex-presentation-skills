# 研習提示範例

## PPTX｜研究摘要

```text
使用 pptx-teaching-deck。
讀取這份研究摘要，先提出 8 頁故事線，
每頁只有一個主張，圖表保留指標、單位與限制。
確認故事線後製作可編輯 PPTX，
渲染所有頁面並修正文字溢出與對齊。
```

## PPTX｜教學單元

```text
使用 pptx-teaching-deck。
把這份教材改成 10 頁教學簡報，
包含問題引入、迷思澄清、比較、案例與帶走句。
每 3 到 5 頁安排一次理解任務。
```

## HTML｜六小時研習

```text
使用 html-slide-deck。
把課程規劃製作成 16:9 HTML 簡報，
支援方向鍵、章節選單、頁碼、講者備註與列印模式。
所有素材放在本地，完成後測試 1280×720 與窄螢幕。
```

## HTML｜研究成果

```text
使用 html-slide-deck。
將三張研究圖表製作成可離線播放的成果簡報，
每張圖表先說結論，再說證據與限制。
不得公開個資，也不要把相關說成因果。
```

## AI 圖片型｜海報式簡報

```text
使用 image-poster-deck。
將這份研習內容製作成 6 頁整頁圖片簡報，
使用 Codex 內建生圖，不使用外部 API Key。
每頁只放一個短句，統一視覺風格，
逐頁檢查中文字後打包成 PPTX。
```

## AI 背景可編輯｜正式教學版

```text
使用 image-editable-deck。
為這份測驗統計簡報生成 8 張無字 AI 背景，
背景必須預留文字區，不能出現文字、數字或標籤。
所有標題、公式、數值與圖表標籤用 PowerPoint 可編輯物件，
完成後渲染並檢查對比與文字溢出。
```
